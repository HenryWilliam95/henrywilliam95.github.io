Ship Happens

Ship Happens is a game where players have to manage their time and must work together in a Co-Operative environment to successfully complete tasks to keep their ship afloat.

The game was developed using Unity3D and C# with a two-man team using agile methods such as JIRA to track task allocation and GitHub for version control. The team were required to pitch the game to a panel of investors every four weeks.
